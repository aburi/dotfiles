## Alex's Dot Files

Storing my dotfiles in one location and symlinking in ordering to make it easier to
keep my files in one place when transferring computers.

```
├── .aliases 				#
├── .bash_profile			# settings for bash shell
├── .bashrc
├── .vimrc
├── .zprofile
├── .zshrc
└── .zshrc.pre-oh-my-zsh
```
